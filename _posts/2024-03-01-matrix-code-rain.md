## 🚀 Vibe Coding Project

这是一个纯粹由灵感驱动的 Vibe Coding 项目，旨在 macOS 上重现并超越经典的《黑客帝国》数字雨效果。

它不仅仅是一个屏幕保护程序，更是一种赛博朋克氛围的营造者。

---

## ✨ 核心亮点

- **沉浸式视觉**: 5层不同流速与景深的代码雨，营造真实的立体空间感。
- **文化融合**: 完美融合 ASCII 字符与《道德经》、《论语》等中文经典，赛博与国风的碰撞。
- **动态交互**: 随机出现的红色“故障”流与周期性的科幻警语（如“系统即将崩溃”），打破单调。
- **极致性能**: 基于原生 Objective-C 与 Cocoa 框架开发，60 FPS 丝滑运行，低 CPU 占用。

---

## 💻 技术栈

- **Language**: Objective-C
- **Framework**: Cocoa (macOS ScreenSaver)
- **Design**: Generative Art
